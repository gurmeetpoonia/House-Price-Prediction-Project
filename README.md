# 🏠 House Price Prediction App

This is a Machine Learning project that predicts house prices based on different features like area, bedrooms, bathrooms, and more.

## 🚀 Live App
👉 https://your-streamlit-link.streamlit.app

---

## 📊 Project Overview
This project uses a **Linear Regression model** trained on housing data to predict prices.

## 🧠 Features Used
- Area (sq ft)
- Bedrooms
- Bathrooms
- Stories
- Parking
- Main Road
- Guest Room
- Basement
- Hot Water Heating
- Air Conditioning
- Preferred Area
- Furnishing Status

---

## ⚙️ Tech Stack
- Python 🐍
- Pandas
- NumPy
- Scikit-learn
- Streamlit

---

## 📈 Model Accuracy
- Linear Regression used
- R² Score approx: 0.6 - 0.7
